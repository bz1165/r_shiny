a_npct_subj()

Purpose
-------
Reusable helper for subject-level categorical summaries in rtables.

What it returns
---------------
One row per current row-group level:
- n (%)

Input
-----
x        : usually USUBJID in the current row/column slice
labelstr : row label provided by summarize_row_groups()
.N_col   : column denominator supplied by rtables

Key idea
--------
Count unique subjects in the current slice, then display:
- xx           when n = 0
- xx (xx.x)    when n > 0

Typical pattern
---------------
basic_table(...) %>%
  split_cols_by("ARM") %>%
  add_overall_col("Total") %>%
  split_rows_by("CATEGORY") %>%
  summarize_row_groups("USUBJID", cfun = a_npct_subj)

Function
--------
a_npct_subj <- function(x, labelstr, .N_col) {
  n <- length(unique(x[!is.na(x)]))

  cell <- if (n == 0L) {
    rcell(0, format = "xx")
  } else {
    rcell(c(n, n / .N_col * 100), format = "xx (xx.x)")
  }

  in_rows(" " = cell, .labels = labelstr)
}

When to reuse
-------------
- Protocol deviations
- TEAE by SOC/PT
- Medical history / concomitant medication count tables
- Any hierarchical subject-count frequency table

Notes
-----
- This is intended for subject-level counting, not event-level occurrence counts.
- If one subject can appear multiple times in the raw data, de-duplicate before tabulation
  at the level you want to count.
- .N_col is provided by rtables to afun/cfun; it represents the column-wise denominator.
